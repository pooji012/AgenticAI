# Product Documentation

The product is a B2C productivity mobile app for tasks, habits, planner views, and calendar sync.

Supported platforms:
- Google Play Android app
- Apple App Store iOS app

Current app versions:
- 3.0.1 latest release
- 3.0.0 previous release
- 2.9.8 stable release

Known product areas:
- Login and Google sign-in
- Weekly Planner
- Task sync across phone and tablet
- Habits page
- Calendar sync
- Dashboard startup performance
- Task history and reporting

High priority signals:
- Crash
- Data loss
- Login failure
- Sync failure

Medium priority signals:
- Performance slowdown
- Pricing complaints
- Feature requests with productivity impact

Low priority signals:
- Praise
- Spam
- General non-actionable comments
